# jshell
* New in Java 9
* Terminal based
* Allows to see results of a line of code

$ jshell
> /help
> /help intro
> /reset
> /exit
> int x = 8
> int y = x + x
> x*3
variable name is given by jshell if we don't give
> $3
we can access a variable like this ($ for varibles name given by  jshell for previous statement)
