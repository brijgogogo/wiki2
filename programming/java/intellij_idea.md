# intellij idea

## shortcuts
* <C-N> : open class in editor
* <C-S-N> : open any file in editor
* <C-mouseClick> : go to declaration

## debugging shortcuts
* <S-F10> : run program
* <S-F9> : debug
* <F8> : step over
* <F7> : step into
* <F9> : resume program
* <S-F8> : step out
* <Alt-F9> : run to cursor
