# zathura
Document viewer

## installation
- pacman -S zathura

## supported documents
- pacman -S zathura-pdf-mupdf
Supports PDF, ePub, OpenXPS

## shortcuts
- man zathura
- <c-r> : invert colors
- a : page-fit
- s: fit width
- r: rotate
- d: side-by-side
- f: highlight links
- +/ : zoom out / in
- Tab: index mode
https://github.com/pwmt/zathura/blob/master/doc/man/_bindings.txt
