# nnn

Tab or Shift+Tab or numbers: cycle through contexts

? : help
/ : search
<C-l> : clear filter or apply last one

' : to to fist non-dir file
. : show/hide hidden files



^p : cp
^v : move


! : spawn shell
] : show prompt


~ : go to home
` : go to root
- : go to last visited dir
, : pin the directory (visited by Bookmark-Key + ,)
<C-n> : nav as you type

b : bookmark key
