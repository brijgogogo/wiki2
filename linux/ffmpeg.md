# fffmpeg
record, convert, stream audio, video

- ffmpeg -help

- ffmpeg -formats
supported formats
- ffmpeg -codecs
supported codecs

- ffmpeg -i input_file.mp3 output_file.wav
Audio conversion
- ffmpeg -i input.mp4 c:a libopus output.ogg
set desired codec using the -c command
