# calibre

calibredb cli:
list
--library-path <path>
-s <search>
add file1, file1..
-d : add even if duplicates

- adding from directories:
calibredb --add * -r
-r, --recurse

- calibre search <search expression>
search expressions: title:robot, author:abc


- poppler for pdftohtml required for pdf to mobi/epub/azw3 format
