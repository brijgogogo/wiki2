# ex commands


## editing via command line
- :3copy5 - copy line 3 to line 5
- :3t5 - same as above
- :3m5 - moves
- :3d - delete
- :3,5m8 - range move
- :-2,+1d - relative range
- :'<,'>sort - visual range sort
- :'<,'>d - visual range delete
- :3s/foo/bar/g - replace on line 3
- :2,5normal A, - append comma on lines 2-5
- @: - repeat last Ex command
